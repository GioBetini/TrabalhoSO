Pegar o conteúdo do site:  (Usar modo RAW ou Dados Brutos), esse site traz um arquivo com informações de acesso de uma página (Tiger_King) no Wikipedia no mês de setembro de 2019. Gravar em um arquivo chamado wiki.json na pasta C:\TEMP (\tmp se for Linux) e fazer uma aplicação java que, na classe principal, no método Main chame uma classe de controle que leia o arquivo, e, apresente a data (timestamp (Sem o 00 do final)) e os views de todos os dias.
